Программа для учёта книг в домашней библиотеке

Работали над проектом:
 Нежурин Андрей - Бэкэнд |

Описание проекта:
Данная программа позволяет вести структурированный учёт по имеющимся у вас книгам.
Начиная от просмотра всей внесённой колекции и заканчивая отслеживанием статуса прочитанности книги.

Написано на C#
